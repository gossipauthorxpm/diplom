Для запуска приложения необходимо:
- Система семейства LINUX
- Установленный python 3.11

Запуск приложения:
1. Распакуйте архив по удобному вам пути
2. В корне откройте cmd 
3. Выполните команду - source venv\bin\activate
4. Выполните команду - pip install psutil
5. Выполните команду - pip install requests
6. Запустите файл main.py командой - python main.py
